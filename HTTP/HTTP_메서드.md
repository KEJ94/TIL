# HTTP API를 만들어보자  
### ```리소스```의 의미를 먼저 알자
- ```회원``` 등록
- ```회원``` 조회
- ```회원``` 수정
- ```회원``` 삭제  

리소스는 ```user``` 이다. API를 설계할 때 /user/{id} 이후 __등록, 조회, 수정, 삭제__ 와 같은 __행위__ 는 HTTP 메서드로 분리 해서 설계하자
