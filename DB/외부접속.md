# MySQL(MariaDB) 외부접속
Ubuntu 16.04 MariaDB

### 접속
```
/var/lib/mysql
sudo mysql -u root -p
use mysql
```

### 외부접속 계정 생성
```
create user '계정아이디'%'identified by '비밀번호';
flush privileges;
```  

### bind-address
```
vi /etc/mysql/mariadb.conf.d/50-server.cnf
# bind-address = 127.0.0.1 주석
```

### 재시작
```
service mysql restart
```
